What are the differences between local storage and session storage?

Difference between two types of web storage such as local storage and session storage is given bellow:
Local Storage: Local storage keeps the data even if the session is over. Data will be retained if the data is not deleted manually.
Session Storage: Session storage keeps the data only for the current session. After the session is over it will automatically delete the data.

What are the differences between global scope and block scope?

Scope in javascript is simply the accessibility of variables, functions or objects. There are two types of scope in javascript as follows:
Block Scope: Variable declared in a curly braces {} is known as block scoped and they can not be accessed outside of this scope. Keywords such as let and const provide this type of scope.
Global Scope: Variable declared inside or outside of the curly braces {} with the keyword var is globally scoped. They can be accessed from anywhere in the code.
